## Objectives

- Providing reliable chatbot for pediatric dental clinic customer support.
- To ensure responses are based on verified pediatric dentistry knowledge (e.g., treatments, procedures, aftercare) using a retrieval-based system.
- Help parents better understand pediatric dental care, preventive measures, and treatment options.
- Implement safeguards for handling sensitive health-related information and avoid misinformation.
